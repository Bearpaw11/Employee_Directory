                                 Employee Directory

This is an employee directory app that was built using React

This uses Random User Generator api to generate 50 random users to the table.

You can search through the users by name in the input box.

You can also sort users by name or city by clicking on the header "name" or "city". You can sort ascending or decending. 

This app uses Bootstrap React for the form and table. Regular css is user for the header and styling.

This page is deployed on netlify at  https://keen-stonebraker-f94cb4.netlify.com/
